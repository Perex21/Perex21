## 🏆 Featured Projects

### 💳 [Stripe Agentic Payments](https://github.com/Perex21/stripe-agentic-payments)

Architected an agent-driven payment orchestration platform that translates natural language intents into executable Stripe workflows. Integrated conversational AI with transaction processing to automate payment creation, checkout generation, and workflow execution. Designed to bridge autonomous decision-making with real-world financial operations.

### 📊 [GST Compliance Agent](https://github.com/Perex21/gst-compliance-agent)

Developed an intelligent compliance automation system for invoice validation, anomaly detection, and regulatory reporting. Leveraged machine learning techniques to identify inconsistencies across financial records while streamlining audit workflows. Built to reduce manual compliance overhead and improve operational transparency.

### 🛡️ [Hybrid Phishing Detection Platform](https://github.com/Perex21/Phishing-Automation)

Engineered a multi-layer threat analysis platform combining machine learning, OCR extraction, URL intelligence, and heuristic detection. Aggregates signals from independent detection engines to identify phishing campaigns, malicious domains, and social engineering attempts. Designed with explainable verdict generation for transparent security assessments.
